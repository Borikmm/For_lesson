Уровень: Финал

Задание: Анализ данных о продажах

Описание задачи:
Ваша задача - разработать программу для анализа данных о продажах товаров. У вас есть набор данных, содержащий информацию о продажах различных товаров в разных магазинах.

Требования:
Ввод данных:

Программа должна уметь считывать данные из внешнего файла (например, CSV).
Предусмотрите возможность ввода данных вручную для тестирования.
Анализ продаж:

Реализовать функционал для анализа данных о продажах, включая общую выручку, средний чек, самый продаваемый товар и т.д.
Фильтрация и сортировка:

Реализовать возможность фильтрации данных по различным параметрам (например, по магазинам, товарам, датам).
Добавить функционал сортировки данных.
Отображение результатов:

Вывести основные результаты анализа, такие как общая выручка, средний чек, самый продаваемый товар и т.д.
Графическое представление данных (дополнительный балл):

Реализовать визуализацию данных, например, построение графиков выручки по времени или доли каждого товара в общей выручке.

Оценка:
Проект будет оцениваться по следующим критериям:

Корректность анализа данных.
Функциональность фильтрации и сортировки.
Качество визуализации данных (если реализовано).
Обработка ошибок ввода данных.

Для генерации самих данных можно воспользоваться существкующими данными либо сгенерировать их через специальные сервисы.
